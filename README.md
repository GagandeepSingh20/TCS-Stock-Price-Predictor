# 📈 TCS Stock Trend Predictor Using Random Forest

This project implements a machine learning-based stock trend prediction system using historical price data of Tata Consultancy Services (TCS) from the Indian stock market.

It predicts whether the stock price will increase or decrease on the next trading day using rolling technical indicators, momentum features, and a Random Forest classifier with walk-forward backtesting.
